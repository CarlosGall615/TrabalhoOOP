# TrabalhoOOP

Projeto acadêmico em Java com tema da Família Addams.

O sistema demonstra:
- classe, atributo, objeto e encapsulamento;
- uso de `enum` para personagens, classes narrativas, mascotes e cenário;
- seleção de um personagem pronto da família pelo usuário;
- associação automática entre personagem e mascote oficial;
- interação entre personagem, mascote e casa durante a batalha;
- saída completa no console para acompanhar toda a simulação.
